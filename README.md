# ゼロ葬代行サービス ランディングページ

第八行政書士事務所のゼロ葬（直葬）代行サービスのSEO最適化ランディングページです。

## ファイル構成

```
zerosou-site/
└── index.html   ← このファイル1つだけでページが完成します
```

## GitHub Pages での公開手順

1. GitHubで新しいリポジトリを作成する（例：`zerosou`）
2. `index.html` をリポジトリにアップロード（またはコミット）する
3. リポジトリの **Settings** → **Pages** を開く
4. **Source** を `Deploy from a branch` に設定する
5. ブランチを `main`、フォルダを `/ (root)` に設定して **Save**
6. 数分後に `https://ユーザー名.github.io/zerosou/` でアクセス可能になります

## 独自ドメインで公開する場合

1. GitHub Pages の Settings → Custom domain に独自ドメインを入力
2. DNS側でCNAMEレコードを `ユーザー名.github.io` に向ける

## ページに含まれるSEO対策

- title・meta description にキーワードを最適化
- Schema.org 構造化データ（Service・FAQPage）→ リッチリザルト対応
- パンくずリスト
- FAQ アコーディオン（Google検索での強調スニペット対応）
- 関連キーワード網羅セクション
